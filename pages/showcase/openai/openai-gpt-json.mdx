# JSON Generator
The JSON Generator node utilizes the OpenAI GPT (Turbo) API to generate text based on provided user and system prompts, and formats this generated text as a JSON object. This can be particularly useful for applications like chatbots, data analysis tools, and automated content generation where responses need to be in JSON format.

<Node id="@buildship/openai-gpt-json" version="1.1.3" />

## Inputs
To use the JSON Generator node effectively, several inputs need to be provided:

1. **System Prompt** (`string`): This prompt instructs the model on how to format the JSON output.
   - **Sample Prompt**:
     ```
     You are a data analyst API capable of sentiment analysis that responds in JSON.  The JSON schema should include: 
     {
         "sentiment_analysis": {
             "sentiment": "string (positive, negative, neutral)",
             "confidence_score": "number (0-1)"
        }
     }
     ```

2. **API Key** (`string`): Your OpenAI API Key. You can obtain your API Key [here](https://platform.openai.com/account/api-keys).

3. **User Prompt** (`string`): The user-specific prompt to guide the JSON generation.

4. **Max Tokens** (`number`): Defines the maximum length (in tokens) of the generated JSON. The default is set to 800.

5. **Temperature** (`number`): Controls the randomness of the generated text. Lower temperatures result in more deterministic outputs, while higher temperatures result in more random completions. Range: `0 to 1`. The default is 1.

6. **Model** (`string`): The specific OpenAI model to use for generating the JSON. Available options include:
   - `gpt-3.5-turbo-0125`
   - `gpt-4-turbo-preview`
   - `gpt-4o` (default)
   - `gpt-4o-mini`

## Output
The node returns a JSON object containing the text generated by the OpenAI GPT API. This JSON object will adhere to the schema defined in the system prompt.

For example, given the sample system prompt mentioned above, the output might look like:
```json
{
    "sentiment_analysis": {
        "sentiment": "positive",
        "confidence_score": 0.95
    }
}
```

This structured output makes it easy to integrate and use within various applications requiring JSON format responses.
